What is a Stack?

"An abstract data structure that follows LIFO (last in, first out)"

How should I conceptualize LIFO?

  • how you'd eat a "stack" of pancakes
  • a call stack, or how you'd manage function invocation
  • in a doc - undo and redo
  • in React, using the history object with routing

Creating the Stack class

          
            class Stack {
              constructor() {
                this.first = null; //like the head in a linked list
                this.last = null; //like the tail
                this.size = 0;
              }
            }
Big-O of Stacks

Insertion and Removal are in Constant time: O(1)

Explanation

Like shift and unshift in linked lists, since insertion and removal are at the last, or "top" of a stack. Similarly, with an array structure, you can have push and pop, as those are the "last" elements inserted in an array, it's easy to find and retrieve them using a stack structure without worrying about re-indexing the whole array.
