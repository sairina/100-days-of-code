What is a Stack?

"An abstract data structure that follows LIFO (last in, first out)"

How should I conceptualize LIFO?

  • how you'd eat a "stack" of pancakes
  • a call stack, or how you'd manage function invocation
  • in a doc - undo and redo
  • in React, using the history object with routing

Creating the Stack class

          
            class Stack {
              constructor() {
                this.first = null; //like the head in a linked list
                this.last = null; //like the tail
                this.size = 0;
              }
            }
Big-O of Stacks

Insertion and Removal are in Constant time: O(1)

Explanation

Like shift and unshift in linked lists, since insertion and removal are at the last, or "top" of a stack. Similarly, with an array structure, you can have push and pop, as those are the "last" elements inserted in an array, it's easy to find and retrieve them using a stack structure without worrying about re-indexing the whole array.

Implementation of a Stack

We can use arrays

  • push/pop (preferred, since we're working from the last ones inserted)
  • unshift/shift

or, we can use linked lists.

Implementing POP in Stack

            
              class Stack {
                constructor(){
                  this.first = null;
                  this.last = null;
                  this.size = 0;
                }

                pop(){
                  if(!this.first) return null;
                  let removed = this.first;
                  if(this.first === this.last){
                    this.last = null;
                  }
                  this.first = this.first.next;
                  this.size--;
                  return removed.value;
                }
              }

Implementing PUSH

            
              class Node {
                constructor(value){
                  this.value = value;
                  this.next = null;
                }
              }

              class Stack {
                constructor(){
                  this.first = null;
                  this.last = null;
                  this.size = 0;
                }

                push(val){
                  let newNode = new Node(val);
                  if(!this.first){
                    this.first = newNode;
                    this.last = newNode;
                  } else {
                    let temp = this.first;
                    this.first = newNode;
                    this.first.next = temp;
                  }
                  return ++this.size;
                }
              }
